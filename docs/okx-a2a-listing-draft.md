## ASP 名称（技术占位）

VALORANT CN Performance Lab

## 一句话

把国服卡顿、低帧与“不跟手”拆成可验证路径，交付带证据、复测和回滚的 Windows 优化方案，不碰反作弊绕过。

## 服务 1：国服 FPS / 卡顿诊断

- **适用**：平均 FPS 低、1% low 差、瞬时卡顿、笔记本性能不稳定。
- **输入**：服别、硬件与 Windows 版本、显示器/刷新率、症状、同场景基线；可选只读诊断 JSON。
- **交付**：瓶颈排序、最多 5 个优先动作、依据等级、风险/条件、复测协议、回滚卡、来源日期。
- **不包含**：远程系统改动、驱动/BIOS 代操作、反作弊修改、FPS 保证。
- **建议定价**：黑客松演示期免费或低固定价；真实价格在登记前根据 OKX.AI 最小值与同类服务刷新。

## 服务 2：ACE / Vanguard 兼容性分诊

- **适用**：ACE CPU 虚拟化弹框、Vanguard 错误码、国服/国际服教程冲突。
- **输入**：准确弹框/错误码、服别、Windows build、Secure Boot/TPM/VBS/HVCI 观察、WSL/Docker/虚拟机与 Windows Hello 依赖。
- **交付**：官方路径匹配、不可做项、条件排查顺序、依赖影响、回滚与官方支持入口。
- **不包含**：绕过、关闭或欺骗反作弊；封禁申诉代理；未知错误的确定性承诺。

## A2A 订单验收条件

1. 明确服别和症状分类，或清楚列出仍缺事实。
2. 每项建议有依据等级、适用条件、观察指标和回滚。
3. 没有 A/B 数据时写“诊断方案”，不宣称“优化完成”。
4. 交付中不出现反作弊绕过、永久降低执行策略或无条件关闭 Windows 安全能力。
5. 来源直接可访问，并标明证据刷新日期。

## 拒单/转介

- 请求作弊、绕过反作弊、隐藏驱动/进程或规避封禁：拒绝并提供合规排障/申诉入口。
- 需要代操作 BIOS、远程管理员权限或写系统设置：本服务只交付方案，需用户本地执行或找可信技术人员。
- 疑似硬件故障、蓝屏或数据风险：停止调参，转厂商/维修/备份路径。
